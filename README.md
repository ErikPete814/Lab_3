# Lab 3 - Student Score Analyzer

## Description

This C++ program simulates a teacher entering the scores of 10 students. The program stores the scores in an array and performs several calculations to analyze the class performance.

## Features

* Accepts 10 student scores (0–99).
* Calculates the total of all scores.
* Calculates the average score.
* Finds the highest score without sorting the array.
* Finds the lowest score without sorting the array.
* Counts how many students passed the class (70 or higher).

## Concepts Used

* Arrays
* Functions
* Loops (`for`)
* Conditional statements (`if`)
* User input (`std::cin`)
* Output (`std::cout`)

## How to Run

1. Compile the program with a C++ compiler.
2. Run the executable.
3. Enter the scores for 10 students when prompted.
4. The program will display:

   * Total score
   * Highest score
   * Lowest score
   * Average score
   * Number of students passing
